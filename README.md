# Diplomatura en Desarrollo Web Frontend

Repositorio que reúne los proyectos realizados durante la Diplomatura en Desarrollo Web Frontend dictada por la Universidad Tecnológica Nacional (UTN) - Facultad Regional Reconquista.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Manipulación del DOM
- Lógica de programación
- Organización modular del código

---

# Proyectos incluidos

## Quidditch Supplies (HTML & CSS)

Landing page temática inspirada en el universo de Harry Potter.

**Conceptos trabajados:**
- Maquetado semántico
- Diseño responsive
- Organización visual con CSS
- Estructuración clara del contenido

---

## Memotest (JavaScript)

Juego interactivo de memoria desarrollado con JavaScript puro.

**Funcionalidades:**
- Generación dinámica de cartas
- Lógica de comparación
- Control de estados del juego
- Manipulación del DOM

---

## Agenda de Tareas (JavaScript Avanzado)

Aplicación para gestión de tareas desarrollada con JavaScript avanzado.

**Funcionalidades:**
- Crear y eliminar tareas
- Manejo dinámico del DOM
- Utilización del localStorage
